1    # My Habit Tracker Application/Project
2    
3    An application that allows users to create, track and analyse their habits. 
4    It is a python project that utilizes SQLite storage and command-line
5    interface (CLI) for interaction
6    
7    ## Features
8    
9    - **Create Habits**: Add new habits with a name and periodicity (daily or weekly).
10   - **Get Completion Dates**: Tracks completion dates for each habit.
11   - **Analyze Habits**:
12     - List all currently tracked habits.
13     - List habits with the same periodicity.
14     - Get the longest run streak for a specific habit.
15     - Get the longest run streak across all habits.
16     - Get the longest streak by periodicity.
17   - **Edit Habits**: Modify the name or periodicity of existing habits.
18   - **Delete Habits**: Remove habits from the tracker.
